Auto generates keyword and description tags for SEO as well as allows the user to specify tags for their homepage.

== Description ==

Auto generates keyword and descriptions tags for SEO using Category, Post, and page information for themes which do not support this by default.  Also allows user to specify keywords and descriptions for every static page on their site.

== Installation ==

1. Upload `auto-keywords-description-generator` directory to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
